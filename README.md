Tugas Eksperimen Operasi Dasar pada Sinyal
Abror Alfarisi Haninda
Projek tentang operasi pada sinyal mulai dari operasi pada sinyal 1D, operasi pada Sinyal 2D, hingga pengujian sistem linier
untuk memulai notebook diperlukan 2 gambar yg bisa diambil dari local directory atau langsung dari dataset yang sudaah ada
Kesimpulan Singkat
Eksperimen ini membuktikan bahwa operasi matematis sederhana (penjumlahan, penggeseran, amplifikasi) adalah fondasi dari Pengolahan Sinyal Digital. 
Pada sinyal 1D, operasi ini digunakan untuk manipulasi amplitudo dan fasa. Pada 2D (citra), prinsip yang sama berlaku namun dalam bentuk matriks 
spasial untuk manipulasi geometri dan warna. Selain itu, eksperimen uji linieritas berhasil memvisualisasikan bagaimana sebuah sistem linier murni 
akan selalu mematuhi hukum homogenitas dan additivitas.
